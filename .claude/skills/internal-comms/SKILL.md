## When to use this skill
To write internal communications, use this skill for:
- 3P updates (Progress, Plans, Problems)
- Company newsletters
- FAQ responses
- Status reports
- Leadership updates
- Project updates
- Incident reports

## How to use this skill

To write any internal communication:

1. **Identify the communication type** from the request
2. **Ask for context** about the specific communication needs
3. **Follow formatting conventions** appropriate for the communication type

## Communication Types

### 3P Updates (Progress, Plans, Problems)
Structure:
- **Progress**: What was accomplished this period
- **Plans**: What's planned for next period
- **Problems**: Blockers, risks, or issues needing attention

### Status Reports
Include:
- Executive summary
- Key accomplishments
- Upcoming milestones
- Risks and mitigations
- Resource needs

### Incident Reports
Include:
- Incident summary
- Timeline of events
- Root cause analysis
- Impact assessment
- Remediation steps
- Lessons learned

### Company Newsletters
Include:
- Engaging headline
- Key announcements
- Team highlights
- Upcoming events
- Call to action
